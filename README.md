# VTB Инвестиционный Симулятор - Игровой симулятор инвестиций на PyGame

Быстрая сводка по проекту
**Что это**: Финальная версия игрового инвестиционного симулятора на PyGame с тематикой ВТБ.
**Основные возможности**: Торговля акциями, облигациями и вкладами, управление портфелем, рыночные события, аналитика доходности.
**Технологии**: Python 3.10+, PyGame 2.5+, система карточек активов, реалистичная симуляция рынка.
**Где посмотреть**: После запуска доступно в отдельном окне приложения.

## Требования
- **Python 3.10+** (проверено на 3.13.2)
- **PyGame 2.5+** - для графического интерфейса
- **SQLite не требуется** - все данные хранятся в памяти во время сессии

## Установка и запуск

### 1. Клонируйте репозиторий и перейдите в каталог проекта:
```bash
git clone https://github.com/HubbatovaEvilina/vtb-simular.git
cd  vtb-simular
```

### 2. Создайте и активируйте виртуальное окружение (рекомендуется):
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

### 3. Установите зависимости:
```bash
pip install pygame
```

Или установите из файла requirements (если есть):
```bash
pip install -r requirements.txt
```

### 4. Подготовка ресурсов
Создайте папку для логотипов (если её нет):
```bash
mkdir logos
```

Добавьте логотипы компаний в папку `logos/`:
- `sber.png` - логотип Сбербанка
- `vtb.png` - логотип ВТБ  
- `tinkoff.png` - логотип Тинькофф

**Рекомендуемый размер логотипов**: 40x40 пикселей

### 5. Запуск приложения
```bash
python main.py
```

Приложение запустится в отдельном окне с разрешением 1200x800 пикселей.

## Игровой процесс

### Начало игры
- **Стартовый баланс**: 10,000 ₽
- **Продолжительность**: 12 недель
- **Лимит сделок**: 10 операций в день

### Доступные активы
#### Акции:
- **Сбербанк (SBER)** - низкий риск, доходность 6.8%
- **ВТБ (VTBR)** - средний риск, доходность 7.5% 
- **Тинькофф (TCSG)** - высокий риск, доходность 5.2%

#### Облигации:
- **Сбер Sb29R** - низкий риск, доходность 13.26%
- **ВТБ Б1-379** - низкий риск, доходность 14.25%
- **Тинькофф 2R** - средний риск, доходность 12.85%

#### Вклады:
- **Сбербанк «Ключевой»** - доходность 18.0%
- **ВТБ «Двойная выгода»** - доходность 26.0%
- **Тинькофф «СмартВклад»** - доходность 15.0%

### Управление
- **ЛКМ** - выбор активов, нажатие кнопок
- **Клавиатура** - ввод количества для покупки
- **Табы** - переключение между типами активов

### Особенности игры
- **Реалистичная симуляция** - цены изменяются с учетом волатильности
- **Рыночные события** - случайные события влияют на котировки
- **Дивиденды и проценты** - регулярные выплаты по акциям и облигациям
- **Визуализация портфеля** - круговая диаграмма распределения активов

## Классы и компоненты

### Основные классы:
- **GameState** - управление состоянием игры и логикой
- **AssetCard** / **VTBAssetCard** - карточки активов с особым оформлением ВТБ
- **Button** / **InputField** / **TabButton** - элементы UI
- **ASSETS** - конфигурация всех доступных активов

### Ключевые функции:
- `main()` - главный игровой цикл
- `_draw_main_screen()` - отрисовка основного интерфейса
- `_draw_news_window()` - окно рыночных новостей
- `update_prices()` - обновление котировок активов
